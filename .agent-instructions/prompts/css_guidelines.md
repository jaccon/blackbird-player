# Prompt: UI and CSS Generation Rules

When creating or updating UI elements:
1. Prioritize Vanilla CSS in `index.css` leveraging variables.
2. Ensure Cyberpunk aesthetic where applicable (neon, dark gradients), but keep structure clean.
3. No React/Vue code unless instructed. BlackBird uses pure DOM manipulation (VanillaJS).
4. Use standard modern layout (Flexbox/Grid). 
5. Ensure responsive traits.
